# How to use 

cargo run -- data/zky.txt
Using 1, 2, 3... select the right answer and at the end of the test, you will learn how you did


# Test format

[Q] = The question name
[A] = Incorrect answer
[C] = Correct answer

The space between the question and the answer is important

Warning: There must not exist an empty line in the test files, otherwise the app will crash, I didn't want to deal with it.
